##苦戦した点、共有したいハマりポイント: -チーズがたらりと垂れるcss(・・・上記①) -画像を透過させるCSSを使用すると、文字まで透過されたため、画像処理ソフトで透過の処理を済ませてimgとして使用。 (・・・上記②)
